# NeuroGarden 隐私、权限与非医疗边界说明

NeuroGarden 面向 AI+疗养场景，定位为“结构化状态观察与温和关怀支持工具”。它不提供医疗诊断，也不能替代医生、心理咨询师或专业机构。

系统只在用户授权后，根据结构化数据观察当前状态是否持续偏离个人日常节奏，并在需要时提供本地提醒或授权守护确认。

## 不保存的数据

- 用户输入原文
- 聊天内容
- 密码内容
- 原始文本
- 完整敏感对话
- 被动采集到的具体语义文本

## 会保存的数据

- 心率
- 呼吸频率
- 运动状态
- 打字速度
- 删除频率
- 停顿时长
- 输入节奏变化
- 天气
- 时间段
- 风险评分
- 异常事件摘要
- 监护人反馈

## 权限用途

### 健康数据权限

用于读取或接收心率、呼吸、运动状态和 Wear OS 数据，帮助判断状态是否相对个人基线发生持续偏离。

### 无障碍服务权限

用于统计打字速度、删除频率、停顿时长和输入节奏变化。系统不通过无障碍服务保存用户输入内容、聊天内容、密码内容或具体语义文本。

### 通知权限

用于本地状态波动提醒、守护确认提醒和照护确认提醒。提醒频率会受到当前使用模式、冷却时间、每日上限和数据可信度限制。

### 网络权限

用于天气数据、Agent API 和可选远程分析。天气 API 或网络不可用时，系统会自动使用 Mock 天气兜底，不影响应用运行。

## Agent 数据边界

允许发送给 Agent 的信息：

- 指标偏离百分比
- 风险等级
- 数据可信度
- 时间段
- 天气
- 监护人反馈摘要

不允许通过被动采集发送给 Agent 的信息：

- 用户输入原文
- 聊天内容
- 私密文本
- 未授权健康原始数据

主动关怀对话属于用户主动输入内容，只用于当前对话生成温和回复。系统应优先保存摘要和标签，避免长期保存敏感全文。

## 非医疗声明

NeuroGarden 不使用“疾病判断”“医疗诊断”“极端行为预测”等表达。界面和答辩中建议使用以下表述：

- 状态偏离
- 节律波动
- 建议关注
- 守护确认
- 照护提醒
- 重点关注

推荐声明：

> NeuroGarden 不提供医疗诊断，也不能替代医生、心理咨询师或专业机构。系统仅根据用户授权采集的结构化数据，辅助观察用户当前状态是否持续偏离个人日常节奏，并在需要时提供本地提醒或授权守护确认。
