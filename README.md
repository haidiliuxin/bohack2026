# NeuroGarden AI 个体化情绪守护与康养关怀系统

NeuroGarden 是一个基于 Android 手机和 Wear OS 智能手表的 AI 个体化情绪守护与康养关怀系统。系统通过长期学习用户自己的心率、呼吸节奏、打字速度、删除频率、停顿时长、手表运动状态和主动反馈，建立个人情绪与身体节奏基线。当当前状态明显偏离个人稳定节奏时，系统会生成风险关注提示、呼吸引导、AI 关怀建议和守护联系人提醒建议，形成“习惯记忆 - 状态偏离识别 - Agent 建议 - 关怀反馈 - 个体化调整”的闭环。

本项目不使用前置摄像头，不做医学诊断，不预测极端行为，仅用于压力感知、情绪支持、康养辅助和人文关怀。

## 技术栈

- Android 手机端：Kotlin、Jetpack Compose、Material 3、ViewModel、Coroutine + Flow、Room、DataStore
- Wear OS 手表端：Kotlin、Compose for Wear OS、MockHeartRateClient、Health Services 预留接口、Wearable Data Layer 预留接口
- 共享模块：SensorPacket、StressResult、EmotionState、TherapyPlan 等数据模型

## 模块结构

- `app`：手机端主应用，包含模拟数据、压力评分、关怀引导、结果页和历史记录
- `wear`：Wear OS 应用，包含手表首页、模拟心率、震动呼吸引导接口
- `shared`：双端共享模型和简单 JSON 工具

## 当前核心功能

- 手机端模拟体验完整闭环
- 平静、紧张、焦虑、运动干扰四种模拟状态
- 压力评分和运动干扰置信度修正
- Canvas 动态情绪地形图
- Compose 呼吸光圈动画
- AI 关怀引导语与关怀策略映射
- Room 保存关怀记录
- Wear OS Mock 心率和触觉反馈接口
- 被动守护前台服务和本地高风险通知
- 多维情绪状态估计和趋势窗口
- 用户轻量标注与近期反馈校准
- 简化关怀模式，适配老人、疲惫用户和低交互场景
- 本地习惯记忆清除

## 主要数据来源

- 心率
- 呼吸节奏
- 打字速度
- 删除频率
- 停顿时长
- 手表运动状态
- 用户主动反馈

## 运行方式

1. 使用 Android Studio 打开 `NeuroGarden` 目录。
2. 等待 Gradle 同步完成。
3. 选择 `app` 配置运行手机端。
4. 选择 `wear` 配置运行手表端或 Wear OS 模拟器。

## 模拟模式演示步骤

1. 打开手机端 App。
2. 点击“进入模拟体验”。
3. 在实时状态页点击“切换模拟状态”，切到焦虑或运动干扰状态。
4. 点击“开始关怀引导”。
5. 观察心率、呼吸、压力评分逐渐下降。
6. 观察情绪地形从山峰逐渐变平，呼吸光圈按节奏缩放。
7. 点击“结束”进入结果页。
8. 点击“保存记录”。
9. 点击“查看历史”，确认记录已保存。

## 被动守护模式

1. 打开“守护模式”。
2. 开启守护提醒并点击“启动被动守护”。
3. 系统会以前台服务方式运行，通知栏显示 NeuroGarden 正在守护。
4. 点击“打开无障碍设置”，手动开启“NeuroGarden 输入节奏特征”。
5. 开启后，系统只统计打字速度、删除频率和停顿时长等特征，不保存或上传输入文本内容。
6. 当连续检测到较高风险状态时，App 会发送本地通知，引导用户进入关怀页。
7. 如果用户不想进行复杂交互，可以在“守护模式”中开启“简化关怀模式”。

被动守护当前仍是比赛演示实现：手机端会结合本地模拟心率/呼吸波动和无障碍输入节奏特征进行风险判断。真实手表被动心率接入需要继续完成 Wear OS Health Services Passive Monitoring。

## 情绪判断准确性改进

当前版本已经从单一压力分升级为多层判断：

- 是否偏离个人基线
- 是否存在运动干扰
- 更可能的状态类型，如高压紧张、焦虑紧绷、低落疲惫、疲惫恢复慢、运动干扰
- 5 分钟、30 分钟、2 小时趋势窗口
- 用户轻量标注：累、烦、低落、紧张、没事
- 有效提醒率和误报反馈
- 近期反馈校准，误报多时降低打扰强度

系统仍然不做医学诊断，只表达“状态偏离”“可能状态”“关怀建议”和“置信度”。

## 隐私声明

NeuroGarden 不使用前置摄像头，不上传原始音频、完整隐私文本或未经处理的敏感健康数据。用户习惯数据默认本地存储，用户可以删除历史记录和习惯样本。Agent 只接收必要特征值，例如心率偏离、呼吸节奏、打字速度、删除频率、停顿时长和用户反馈标签。

NeuroGarden 仅用于压力感知、情绪支持、呼吸训练和康养关怀，不构成医学诊断或治疗建议。

## 答辩表述

NeuroGarden 是一个 AI 个体化情绪守护与康养关怀系统。它不试图诊断用户，而是学习每个人自己的日常节奏，在状态偏离时提供温和、可解释、可反馈的关怀建议，并通过用户反馈不断调整个人化阈值。
